# Imágenes AR

Coloca aquí las 6 imágenes que quieres mostrar en AR:

## Marcadores HIRO (primeros 3):
- **ar-image-1.jpg** - Se mostrará con marcador HIRO
- **ar-image-2.jpg** - Se mostrará con marcador HIRO
- **ar-image-3.jpg** - Se mostrará con marcador HIRO

## Marcadores KANJI (siguientes 3):
- **ar-image-4.jpg** - Se mostrará con marcador KANJI
- **ar-image-5.jpg** - Se mostrará con marcador KANJI
- **ar-image-6.jpg** - Se mostrará con marcador KANJI

## ⚠️ IMPORTANTE:
Como todos los HIRO usan el mismo marcador físico, el sistema mostrará UNA imagen aleatoria de las 3 cuando detecte HIRO.
Lo mismo aplica para KANJI: mostrará UNA de las 3 imágenes KANJI.

## Recomendaciones:
- Formato: JPG o PNG
- Tamaño recomendado: 512x512 o 1024x1024 píxeles
- Peso: Menos de 500KB para mejor rendimiento
- Contenido: Cualquier imagen que quieras (logos, productos, personajes, etc.)
